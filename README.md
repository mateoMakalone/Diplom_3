# Diplom_3
Покрытие автотестами UI сервиса Stellar Burgers
---
Technology stack:
Java 11,
Junit 4.13.1,
GSON 2.8.9,
Maven 3.8.1,
javafaker 1.0.2,
lombok 1.18.20,
rest-assured 4.4.0,
Allure 2.15.0,
selenium-api 3.141.59
Для создания отчета Allure нужно выполнить комнаду "mvn clean test".
Для визуализации отчета используйте команду "mvn allure:serve".
